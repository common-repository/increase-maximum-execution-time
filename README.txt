Plugin Details:

Name: Increase Maximum Execution Time
Contributors: Deepak Kaushik
Tags: Increase Maximum Execution Time to 10 min
Requires at least: 3.0
Tested up to: 5.1.1 
Stable tag: max_execution

Description:

This plugin is used for increase wordpress Maximum Execution Time to execute long query and data manipulation.

Plugin Functionality: 

1) Activate from admin that automatically increase max execution time 10min

Installation: 

1) Upload `Increase Maximum Execution Time` to the `/wp-content/plugins/` directory
2) Activate the plugin through the 'Plugins' menu in WordPress

Upgrade notice:

Automatic updates should work great for you.
